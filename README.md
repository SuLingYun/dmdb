# DMDB - 达梦数据库快速恢复工具

达梦数据库（DM）快速恢复脚本，支持同机/异机快速恢复，自动显示可恢复时间范围。

## 📋 功能特性

- ✅ **自动检测备份集** - 扫描备份目录，识别有效的备份集
- ✅ **可恢复时间范围** - 显示完整的时间线，支持 PITR（时间点恢复）
- ✅ **同机/异机恢复** - 灵活支持不同场景的恢复需求
- ✅ **自动执行恢复** - 一键完成数据库恢复流程
- ✅ **详细日志记录** - 完整的操作日志，便于问题排查
- ✅ **超时控制** - 可配置的 dmrman 执行超时时间

## 🚀 快速开始

### 环境要求

- 达梦数据库 DM8+
- Bash Shell
- 需配置正确的数据库环境变量

### 使用方法

```bash
# 1. 克隆仓库
git clone https://github.com/SuLingYun/dmdb.git
cd dmdb

# 2. 修改配置（根据实际环境）
vim dm_recover.sh

# 3. 执行恢复脚本
bash dm_recover.sh
```

### 配置说明

脚本配置区位于文件头部：

```bash
# 数据库连接配置
DB_USER="SYSDBA"           # 数据库用户名
DB_PASS="your_password"    # 数据库密码
DM_HOME="/data/dm"         # DM 安装目录
DM_DATA="/data/dmdata"     # 数据文件目录
DM_BAK="/data/dmbak"       # 备份文件目录
DM_ARCH="/data/dmarch"     # 归档日志目录
DB_SERVICE="DmServiceDM"   # 服务名称
DB_PORT="5236"             # 端口号

# 恢复后是否自动执行全量备份
AUTO_BACKUP="no"
```

## 📝 脚本说明

### 恢复流程

1. **环境检查** - 验证 DM 环境变量和工具可用性
2. **备份集扫描** - 查找所有有效的备份集
3. **时间线显示** - 展示可恢复的时间范围
4. **用户确认** - 提示用户选择恢复时间点
5. **执行恢复** - 调用 dmrman 执行恢复操作
6. **启动验证** - 启动数据库服务并验证恢复结果
