# DM-Database-Recovery

达梦数据库快速恢复工具集

## 概述

本仓库提供达梦数据库（DM Database）的快速恢复脚本，支持同机/异机场景下的数据库恢复操作。

### 主要功能

- 快速恢复：支持全量备份 + 增量备份链的自动恢复
- 时间点恢复：可精确恢复到指定时间点
- 归档恢复：基于归档日志的精细化恢复
- 进度可视化：实时显示恢复进度、速度、剩余时间
- 自动校验：备份集完整性检查
- 安全机制：恢复前自动备份当前数据

## 脚本说明

| 脚本 | 说明 |
|------|------|
| `dm_recover.sh` | 达梦数据库快速恢复主脚本 |

## 快速开始

### 前置条件

- 达梦数据库已安装（DM_HOME 已配置）
- 已配置归档模式
- 存在有效的备份集

### 执行恢复

```bash
chmod +x dm_recover.sh
./dm_recover.sh
```

脚本提供三种恢复模式：

1. **恢复到最新状态** - 推荐选项，自动应用所有增量备份和归档日志
2. **恢复到指定时间点** - 精确恢复到某个时间点
3. **仅恢复备份** - 仅恢复数据文件，不应用归档

## 配置说明

脚本头部提供可配置参数：

```bash
DB_USER="SYSDBA"              # 数据库用户名
DB_PASS="your_password"       # 数据库密码
DM_HOME="/data/dm"            # 达梦安装目录
DM_DATA="/data/dmdata/DMTEST" # 数据目录
DM_BAK="/data/dmbak/..."      # 备份目录
DM_ARCH="/data/dmarch/..."    # 归档目录
DB_SERVICE="DmServiceDMTEST" # 数据库服务名
DB_PORT="5236"                # 数据库端口
```
